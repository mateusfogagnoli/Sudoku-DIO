# Sudoku UI — Quick Start

Rápido: compile e abra a interface gráfica.

Requisitos:

1) Abra PowerShell na pasta do projeto.

2) Execute:

```powershell
.\run.ps1
```

3) Se o PowerShell bloquear scripts, rode antes:

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
.\run.ps1
```

Pronto — a janela do jogo deve abrir com um puzzle de exemplo.

Arquivo principal da UI: `UIMain`
